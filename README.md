# Bright Future U Static Website

Static HTML, CSS, and JavaScript implementation of the Bright Future U website.

## Preview

![Bright Future U home page preview](assets/Home.png)

## Pages

- `index.html` - Home page
- `about-us.html` - About Us page

## Local Preview

Run a static server from the project root:

```bash
python3 -m http.server 4173
```

Then open:

```text
http://127.0.0.1:4173/
```
